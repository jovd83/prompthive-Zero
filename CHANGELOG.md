# Changelog

## [1.0.0] - 2025-12-30

### Added
- **Local Storage**: Implemented File System Access API integration.
- **Single File Build**: Configured Vite to bundle everything into `index.html`.
- **Dashboard**: Main view with Prompt Grid and Sidebar.
- **Prompt Management**: Create, View, and Edit prompts.
- **Collections**: Create and view collections.
- **Attachments**: Support for uploading and viewing images (stored in `uploads/`).
- **Dark Mode**: Default dark theme matching original Prompthive.
