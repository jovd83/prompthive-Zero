# User Manual - Prompthive Zero

## Workspace Setup
When you first open Prompthive Zero, you will be asked to select a Project Folder.
1. Click **Select Project Folder**.
2. Choose a dedicated empty folder on your computer (e.g., `Documents/MyPrompts`).
3. The app will ask for **View** and **Save** permissions. Click **Allow**.
   _Note: You must do this every time you open the app due to browser security._

## Creating a Prompt
1. Click the **+ New Prompt** button in the top right.
2. Enter a **Title** and **Body**.
3. Add **Tags** (comma separated) to help find it later.
4. **Drag and drop images** onto the upload area to attach screenshots.
5. Click **Save**.

## Using Collections
1. In the sidebar, click the **+** button next to "Collections".
2. Enter a name for your new collection.
3. (Future) Drag prompts into collections or select a collection when creating a prompt.

### Advanced Features
- **Nested Collections**: Create sub-collections by creating a collection while another is selected.
- **Import/Export**: Click "Data Management" in the sidebar to backup or restore your library.
- **Advanced Fields**: When editing a prompt, click "Advanced Settings" to add Examples or Instructions.
- **Tag Filtering**: Click any tag on a prompt card to filter your view by that tag.
